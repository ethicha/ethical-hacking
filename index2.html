5.sqlmap

<>
sqlmap, nah untuk tools satu ini juga sering mimin pake untuk inject website yang vuln / error, ya pasti sebagian orang yang baru mulai cyber security itu juga pasti tau soalnya ini udh mendunia banget sama seperti tools yang lainnya tadi, tapi kalau kalian belum faham biar mimin jelaskan, tertulis dari website kali.org bahwa "Tujuan sqlmap adalah untuk mendeteksi dan memanfaatkan kerentanan injeksi SQL dalam aplikasi web.
